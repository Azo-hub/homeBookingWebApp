## About the Project :link:
The following technologies and solutions were present in this project as highlighted below:
- Java 17;
* Spring Boot version 3.1.0;
+ Spring Security 6;
- Spring Data JPA;
+ RestApi;
* Hibernate;
- Postgre;
+ Cloudinary;
- Gmail Smtp;
* Jwt Token;
+ Docker;
